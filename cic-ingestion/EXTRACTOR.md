# Ingestion Extractor Lessons

Lessons learned from DLQ replays and verification failures in the ingestion layer.
Entries are auto-appended when a reasonCode reaches replayCount >= 2.

## How to Use

- Search by reasonCode (MISSING_FIELD, BAD_TYPE, EXTRACTOR_ERROR) to find patterns
- Use "Fix" and "Why" sections to decide on root cause vs. quick workarounds
- Link to specific DLQ replay dates via logDir timestamps
