    Rewrite Labs PRD — Phase 1.0

    1. Product Summary

    Rewrite Labs is an AI‑powered website redesign engine that automates discovery, extraction, redesign, outreach, and optimization.

    Phase 1.0 introduces:

    • Multi‑variant redesign
    • Closed‑loop optimization
    • Predictive routing
    • Self‑healing CIC layer
    • Enterprise‑grade API

    2. Goals & Non‑Goals

    Goals

    • Automate redesign generation
    • Improve redesign quality via CLOE
    • Enable enterprise integrations
    • Provide deterministic, governed infrastructure

    Non‑Goals

    • Full CMS replacement
    • Custom development services
    • Hosting infrastructure

    3. User Personas

    SMB Owner

    Wants a modern website without complexity.

    Agency Partner

    Wants scalable redesign output.

    Enterprise Web Team

    Wants automation, governance, and integration.

    4. Functional Requirements

    4.1 Discovery

    • Crawl websites
    • Score outdatedness
    • Classify industry

    4.2 Extraction

    • Parse DOM
    • Identify semantic blocks
    • Normalize content

    4.3 Redesign Engine 2.0

    • Generate 3–10 variants
    • Apply industry patterns
    • Score variants

    4.4 Outreach Engine

    • Personalized messaging
    • Multi‑touch sequences
    • A/B testing

    4.5 CLOE Optimization

    • Variant scoring
    • Prompt tuning
    • Extractor tuning

    4.6 CIC Governance

    • Drift detection
    • Lineage enforcement
    • Routing validation
    • Self‑healing

    5. Technical Requirements

    Inference

    • Nemotron Nano 30B
    • NIM Gateway batching
    • GPU acceleration

    API

    • REST v1
    • OAuth2
    • Rate limiting

    Security

    • Zero‑trust routing
    • RBAC
    • Audit logs

    6. KPIs

    • Redesign quality score
    • Outreach conversion rate
    • Variant performance
    • Latency (p99)
    • Drift incidents
