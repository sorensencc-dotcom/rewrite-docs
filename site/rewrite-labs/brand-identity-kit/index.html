    Rewrite Labs Brand Identity Kit

    Colors • Typography • Layout • Visual System

    Brand Colors

    Primary Palette

    • RL Blue — #1A73E8 (Primary action, headings)
    • RL Black — #0A0A0A (Text, dark backgrounds)
    • RL White — #FFFFFF (Light backgrounds, contrast)

    Secondary Palette

    • Accent Green — #00FF88 (Success, highlights)
    • Soft Gray — #F5F5F5 (Backgrounds, borders)
    • Slate — #2E2E2E (Secondary text)

    Usage

    • RL Blue: CTAs, buttons, links
    • RL Green: Success states, confirmations
    • RL Slate: Secondary copy
    • RL Gray: Backgrounds, dividers

    Typography

    Headings

    • Typeface: Inter Bold
    • Style: Clean, geometric, modern
    • Sizes: 48px (H1) / 36px (H2) / 24px (H3) / 18px (H4)

    Body

    • Typeface: Inter Regular
    • Size: 16px (body) / 14px (small)
    • Line height: 1.6x
    • Color: #0A0A0A

    Code / Technical

    • Typeface: JetBrains Mono
    • Size: 13px
    • Color: #1A73E8

    Layout System

    Grid

    • 12-column responsive grid
    • 24px gutters
    • 96px max content width on desktop
    • 16px gutters on mobile

    Spacing Scale

    • 8px baseline
    • 8 / 16 / 24 / 32 / 48 / 64 / 96

    Components

    • Rounded corners: 6px (buttons, cards)
    • Shadows: soft, minimal (4px blur)
    • High contrast for accessibility

    Imagery

    • Style: Clean, minimal, modern
    • Backgrounds: Light, high key
    • Subject: Technology, people, redesigns
    • Avoid: Stock photo clichés, busy backgrounds
    • Screenshots: Real product, not mocks

    Logo Usage

    Sizing

    • Minimum size: 120px width
    • Minimum padding: 24px on all sides

    Backgrounds

    • RL Blue logo on light backgrounds
    • RL White logo on dark backgrounds (#1A73E8 or darker)

    Orientation

    • Never stretch or distort
    • Never rotate
    • Never change colors

    Voice & Tone

    • Clear: No jargon, direct messaging
    • Confident: We know what we're doing
    • Helpful: Guiding, not pushy
    • Modern: Current, forward-looking
    • Technical but friendly: Explain the why

    Messaging Pillars

    1. Modernization: Bring outdated websites into modern era
    2. Automation: Redesigns in minutes, not months
    3. Intelligence: AI that learns and improves
    4. Scale: Thousands of redesigns per day

    Words We Use

    • Modern
    • Fast
    • Automated
    • Intelligent
    • Clean
    • Conversion-focused
    • Industry-aware

    Words We Avoid

    • Complicated
    • Magic
    • Black box
    • Overly technical jargon
    • Buzzwords without meaning
